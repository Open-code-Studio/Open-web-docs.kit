# 2026-07-13

## 新增首页小组件系统 (Widgets)
- 创建 `docs/_widgets.md`，单个 MD 文件定义所有小组件及其布局
- 参考 MC Wiki 首页设计，支持五种小组件类型：featured、news、stats、versions、links
- 每种类型有独特的视觉呈现：
  - **featured**: 大卡片，丰富的文字内容 + CTA 按钮
  - **news**: 日期标签 + 新闻条目列表
  - **stats**: 表格形式的统计数据展示，数值列右对齐加粗
  - **versions**: 平台版本网格卡片
  - **links**: 药丸形快速链接列表
- 每个小组件 = `<div class="widget-panel widget-panel--TYPE widget-panel--COLOR">`，通过 `grid-area: AREA` 定位
- 布局由 `<!-- areas:desktop/tablet/mobile="..." -->` 定义，使用 CSS Grid Areas 原生命名区域
- `matchMedia` 监听媒体查询变化，窗口缩放时自动切换桌面/平板/手机布局模板
- 五种 color 变体（primary/secondary/tertiary/error/neutral），头部/边框颜色跟随变化
- 小组件头部带 2px 彩色底边 + emoji 图标 + 标题
- 可选底部 CTA 链接（箭头动画）
- 入场动画 `widgetSlideUp`，hover 时抬升 + 阴影
- 移动端自动 collapse 为单列
- `config.json` 新增 `home.widgets` 配置节点（enabled/file/header），可独立开关
